'pull'
git pull origin master

'push'
git init 디렉토리 초기화
git status 깃 저장소 현재 상태
git pull origin master 지정 해놓은 Repository 에서 자료를 끌어온다.
git status 현재 Git 과의 상태를 Check
(폴더 내부의 빨간색으로 확장자가 적혀있다면 트래킹 되어있지 않다는 것을 표시)
git add. 변경된 파일들을 모두 Tracking!
(빨간 확장자들을 모두 한 번에 add로 추가 가능)
git commit -m "message" 변경사항을 업로드 했을 때 나에게 알려주는 commit 다른 개발자에게 달 수 있는 Commit
git push origin + master 실제로 Git 에 push ! 요청
